# CHEX QUEST 3 DOOM MODDING COMPATIBILITY PATCH
The Chex Quest 3 Doom Modding Compatibility Patch is a work-in-progress mod for Chex Quest 3, Doom, and/or Doom 2, which aims to make Chex Quest 3 compatible with Doom and mods for Doom.
This README file is not planned to be the one shipped with the mod's full release, but still aims to provide documentation to anyone looking for it.
This repository currently only includes code, not assets. Placeholder assets will be added once designed.
